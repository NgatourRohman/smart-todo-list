# 📋 Smart To-Do List

A modern, feature-rich to-do list application with dark mode, local storage persistence, and beautiful notifications.

![Smart To-Do List Screenshot](screenshot.png)

## ✨ Features

- **Task Management**
  - Add new tasks
  - Mark tasks as complete/incomplete
  - Delete tasks
  - Filter tasks (All/Active/Completed)
  - Real-time task counter

- **Dark Mode** 🌙
  - Toggle between light and dark themes
  - Theme preference saved in localStorage
  - Smooth transition animations

- **Smart Notifications** 🔔
  - Success notifications for task additions
  - Error notifications for invalid actions
  - Delete confirmations
  - Auto-dismiss after 3 seconds
  - Manual close option

- **Data Persistence** 💾
  - Automatic save to localStorage
  - Data persists after browser refresh
  - No database required

- **Responsive Design** 📱
  - Works perfectly on desktop, tablet, and mobile
  - Touch-friendly interface
  - Optimized for all screen sizes

## 🛠️ Technologies Used

- **HTML5** - Semantic markup
- **CSS3** 
  - CSS Custom Properties (Variables)
  - Flexbox & Grid layouts
  - Mask-image for icons
  - Smooth transitions
  - Dark mode theming
- **JavaScript (ES6+)**
  - Local Storage API
  - Custom events
  - Dynamic DOM manipulation
  - Modular functions
- **Google Fonts** - Manrope & Poppins
